## Acknowledgements

### Software Frameworks

**MOOS-IvP**
- Developed by: Michael Benjamin, Henrik Schmidt, and the MOOS-IvP community at MIT
- Website: [moos-ivp.org](https://oceanai.mit.edu/moos-ivp/)
- License: GPL
- Used for: Autonomous behavior framework and mission control

**ArduPilot**
- Developed by: ArduPilot Dev Team and community
- Website: [ardupilot.org](https://ardupilot.org/)
- License: GPL v3
- Used for: Flight control and sensor fusion

**MAVSDK**
- Developed by: Dronecode Foundation and contributors
- Website: [mavsdk.mavlink.io](https://mavsdk.mavlink.io/)
- License: BSD 3-Clause
- Used for: MAVLink communication library

### Hardware Platforms

**CubeOrange/CubeOrange+**
- Manufacturer: Hex/ProfiCNC
- Website: [cubepilot.org](https://cubepilot.org/)
- Used for: Flight control hardware

**Skywalker X8**
- Manufacturer: Skywalker
- Platform: Fixed-wing UAV airframe
- Used for: Primary UAV platform

**RFD900x**
- Manufacturer: RFDesign
- Website: [rfdesign.com.au](https://rfdesign.com.au/)
- Used for: Long-range telemetry and mesh networking

**Raspberry Pi**
- Manufacturer: Raspberry Pi Foundation
- Website: [raspberrypi.org](https://www.raspberrypi.org/)
